Will Google crawl these links?

Yes. All the platforms below (Google Sites, Blogger, Medium, WordPress, Tumblr, Telegraph, rentry, github.io) are big, high-authority domains that Google re-crawls constantly — so a public post there gets discovered and the link picked up on its own, usually within days to ~2 weeks. You do NOT need to submit them anywhere.
  • Dofollow links (Google Sites, Blogger, rentry, github.io) pass real ranking power to our pages.
  • Nofollow links (Medium, WordPress, Tumblr, Telegraph) mainly help with fast indexing, referral traffic, and a natural-looking link profile — still worth having, just less direct ranking weight.
  • To help Google find a post faster: make sure it's public and search-engines are allowed (don't tick any "hide from search engines" box). That's all that's needed.

Account & IP rules (read before posting)

  1. One account = one IP. Each platform account should be created and used from its OWN IP / device. Two accounts from the same IP look connected.
  2. IP country does NOT need to match the language. A German article can be posted from any-country IP. What matters is account hygiene, not geo-matching.
  3. Keep one account on one steady IP. Do NOT log into the same account from different countries — Google/Medium will lock it. Pick an IP per account and stick to it.
  4. Drip, don't blast. 1–2 posts per platform per day, spread over 1–2 weeks. A sudden burst of links looks unnatural.
  5. Vary everything. Different persona name, different title, and a DIFFERENT anchor phrase every time (never reuse the exact same anchor). Rotate which page you deep-link.
  6. Always PUBLIC + search-engines allowed. Publish public and do NOT tick any 'hide from search engines' box — Google must be able to find it.
  7. Match language to the geo pages. Post the German article only with the German (ozem-plus.store) links, Dutch with /nl/ links, etc. Don't translate — use the ready text below.
