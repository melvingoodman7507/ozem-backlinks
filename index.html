2 · Rules (read once)

  1. 1 article = 1 platform. NEVER post the same article on two sites. Google keeps only one copy and ignores the duplicates, so those links are wasted. Need another post? Change the title and rewrite the first paragraph — that counts as a new article.
  2. 1 account = 1 IP. Create and use each account from its own IP / device. The IP country does NOT need to match the article language (a German article can be posted from any country's IP). But keep one account on ONE steady IP — don't log into the same account from different countries or it gets locked.
  3. Every article gets exactly 3 links. Use the link map for that geo (below): always include the Homepage link + 2 others. One link on one phrase. Use the exact anchor text shown.
  4. Drip — don't blast. Max 2–3 posts per day, spread across the whole week. Each post from a different account / IP.
  5. Always PUBLIC + search engines allowed. Publish public and do NOT tick any 'hide from search engines' box, or Google can't find it.
  6. Match the language. German article → German (ozem-plus.store) links. Dutch → /nl/ links. Italian → /it/. Polish → /pl/. Never mix.
